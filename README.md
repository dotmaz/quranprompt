# QuranPrompt

An AI-powered quran memorization app that allows you to say what you'd like to listen to, with a pause button that *doesn't* lose your place.

Use it for free: https://quranprompt.vercel.app/


## To run locally

1. Navigate to the `backend` folder
2. Create a `.env` file containing `OPENAI_API_KEY` with your OpenAI API key
3. Run `npm i`
4. Run `npm start`
5. Navigate to the `frontend` folder
7. Run `npm i`
8. Run `npm start`
9. Open `locahost:3000` in a browser
